# Mechanical Service Website

A modern mechanical service web app built with vanilla HTML/CSS/JS.

- **Frontend**: Hosted on GitHub Pages
- **Backend**: Firebase (Firestore, Storage, Analytics)

## Live Site
> https://YOUR_USERNAME.github.io/mech/

## Features
- Hero image carousel (managed via Firebase Storage)
- Services grid loaded from Firestore
- Contact form saving to Firestore
- Admin image upload panel
- WhatsApp floating button
